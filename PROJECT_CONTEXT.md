Sharp AI - Project Context

Project Name

Sharp AI

Vision

Sharp AI is an AI-powered Content Studio SaaS that helps creators, students, entrepreneurs, marketers, and businesses transform a single idea into multiple forms of content quickly and efficiently.

The long-term goal is to create a powerful content-generation platform that combines AI writing, content repurposing, workflow automation, and productivity tools into one modern SaaS application.

Development Rules

When modifying code:

1. Preserve existing functionality whenever possible.
2. Keep code modular and maintainable.
3. Prioritize mobile responsiveness.
4. Avoid unnecessary dependencies.
5. Use clean semantic HTML.
6. Keep CSS organized and scalable.
7. Optimize performance where possible.
8. Maintain a professional SaaS appearance.
